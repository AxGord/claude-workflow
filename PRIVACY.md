# Privacy Policy

**Claude Workflow** does not collect, transmit, or store any personal data.

All data — workflow definitions, session state, and context — is stored locally on your machine in `~/.claude/workflow-state/` and never leaves your device.

The plugin makes no network requests. The web dashboard runs on `localhost` only and is not accessible externally.

## Third-party services

None. This plugin has no external dependencies that process user data.

## Contact

If you have questions about this policy, open an issue at https://github.com/AxGord/claude-workflow/issues.
